-- WHAT ANNOTATION DOES --
Converts RLE ship masks in CSV to YOLO .txt bounding box files.
One .txt file per image, one line per ship.
YOLO format: class x_center y_center w_norm h_norm (normalised 0-1)
Background images (no ships) get EMPTY .txt files.
One label file is SHARED by both SAR and optical (same scene).

-- CONFIGURATION --

RLE decode size    : 768x768 (original Airbus)
YOLO normalise     : 320x320 (Phase 1 output)
Min box size       : 2 px

-- ANNOTATION PIPELINE --
  Step 1: Read EncodedPixels from CSV
  Step 2: Decode RLE -> flat array (height x width)
  Step 3: Reshape -> 2D binary mask (column-major / Fortran order)
  Step 4: Find connected components (each blob = one ship)
  Step 5: cv2.boundingRect() -> pixel-space (x, y, w, h)
  Step 6: Scale from RLE space (768) to target space (320)
  Step 7: Normalise to YOLO format [0, 1]
  Step 8: Write .txt file
